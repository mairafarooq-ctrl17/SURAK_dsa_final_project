## 📌 Project Description

The Stock Price Analyzer is a Python-based program that analyzes stock price data using algorithms. It allows users to view data, sort, search, calculate averages, and find the best profit period.

---

## 🔧 Function Contributions

### 1) calculate_prefix_sum_prices(stocks) — Prefix Sum  
**Done by: Aleena Abid**  
Calculates cumulative stock prices to improve efficiency.

---

### 2) binary_search_by_date(stocks, target_date) — Binary Search  
**Done by: Irtaza Haider**  
Searches stock data quickly by date.

---

### 3) merge_sort_by_price(stocks) — Merge Sort  
**Done by: Irtaza Haider**  
Sorts stock prices from lowest to highest.

---

### 4) calculate_moving_average(stocks, window_size) — Moving Average  
**Done by: Maira Farooq**  
Calculates average price over a given window.

---

### 5) kadane_max_profit(stocks) — Maximum Profit  
**Done by: Maira Farooq**  
Finds best days to buy and sell for maximum profit.

---

## 🎯 Conclusion

This project demonstrates how core algorithms like sorting, searching, prefix sums, and dynamic programming can be applied to real-world stock data analysis.
